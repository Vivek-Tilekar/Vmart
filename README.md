# Java Web Application (JSP, Servlet, JDBC, MySQL)

A dynamic **Java Web Application** built using **JSP, Servlets, JDBC, and MySQL**.  
This project demonstrates how to develop a full web application with server-side logic, database connectivity, and dynamic web pages using Java web technologies.

---

## 🚀 Features

- User-friendly web interface
- Dynamic pages using **JSP**
- Business logic handled by **Java Servlets**
- Database connectivity using **JDBC**
- Data stored and managed using **MySQL**
- CRUD Operations (Create, Read, Update, Delete)
- Form validation
- Server-side request handling

---

## 🛠️ Technologies Used

- **Java**
- **JSP (Java Server Pages)**
- **Servlet**
- **JDBC**
- **MySQL**
- **HTML**
- **CSS**
- **Apache Tomcat**

## ⚙️ Requirements

Before running the project, make sure you have the following installed:

- Java JDK 8 or higher
- Apache Tomcat Server
- MySQL Server
- MySQL Workbench (optional)
- Eclipse / IntelliJ IDEA / NetBeans
- Web Browser (Chrome, Edge, Firefox)

---

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

---

### 2. Import the Project into IDE

Open your IDE and import the project.

Example (Eclipse):

```
File → Import → Existing Projects into Workspace
```

---

### 3. Configure Apache Tomcat

1. Download and install Apache Tomcat
2. Add Tomcat server to your IDE
3. Deploy the project to the Tomcat server

---

### 4. Setup MySQL Database

Open MySQL and create a new database:

```sql
CREATE DATABASE project_db;
```

Import the SQL file provided in the project to create required tables.

---

### 5. Configure Database Connection

Update the JDBC connection details in your Java code:

```java
String url = "jdbc:mysql://localhost:3306/project_db";
String username = "root";
String password = "your_password";
```

---

## ▶️ Running the Application

1. Start the **Apache Tomcat Server**
2. Deploy the project
3. Open your browser and go to:

```
http://localhost:8080/ProjectName
```
